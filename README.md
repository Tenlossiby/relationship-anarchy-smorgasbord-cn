## 📖 项目简介

关系安那其自助拼盘（RA Smörgåsbord）是一个帮助人们探索和定义关系的工具。你与对方可以从任意"菜单"中挑选任意数量的"菜品"，无论是一大份，还是只选一点点。你们共同选择的那些"菜品"，就是你们的关系。

### 💡 核心理念

- **互动沟通**：这个工具高度依赖于关系中的互动和沟通，它不会独立于关系而存在
- **共同定义**：不同的人对同一个词汇可能有不同的定义，没有标准答案，只有你们共同定义的答案
- **非预设框架**：关系不需要符合任何传统或预期的模式
- **开放对话**：通过 28 个维度的卡牌讨论，深入探索彼此的期待
- **灵活调整**：关系是动态的，你可以随时调整你们的"拼盘"
- **包容多元**：支持各种类型的关系和多元之爱

## ✨ 功能特性

## 📖 English Introduction

**Relationship Anarchy Smörgåsbord (RA Smörgåsbord)** is an inclusive tool for exploring and defining relationships. You and your partner can choose any number of "dishes" from any "menu" - whether a large portion or just a little bit. The "dishes" you choose together become your relationship.

### 💡 Core Values

- **Interactive Communication**: This tool relies on interaction and communication within relationships, it cannot exist independently of relationships
- **Co-created Definitions**: Different people may have different definitions for the same term, there is no standard answer, only the answer you define together
- **No Prescribed Framework**: Relationships don't need to fit any traditional or expected patterns
- **Open Dialogue**: Explore each other's expectations through card discussions across 28 dimensions
- **Flexible Adjustment**: Relationships are dynamic, you can adjust your "smörgåsbord" at any time
- **Inclusive & Diverse**: Support all types of relationships and non-monogamy

### 🎯 Key Features

- **28 Relationship Categories**: Covering daily life, emotional support, commitment levels, sexual boundaries, and more
- **Multiple Profiles**: Create independent relationship profiles for different people
- **Asynchronous Collaboration**: Export and import profiles for collaborative filling
- **Smart Comparison**: Compare profiles to discover consensus and differences

### 🎨 User Experience

- **Privacy First**: All data stored locally in your browser, no uploads
- **Responsive Design**: Perfect support for mobile, tablet, and desktop devices
- **Dark Mode**: Support for light, dark, and system themes
- **Smooth Interaction**: Clean and intuitive interface with warm Morandi green colors

### 🔐 Data Security

- 🔒 All data stored only in your browser locally
- 🔒 No personal information collected
- 🔒 Export backup support to prevent data loss
- 🔒 Delete profiles at any time

## 🚀 使用指南

### 在线访问

🌐 **访问地址**：[https://relationship-anarchy-smorgasbord-cn.netlify.app/](https://relationship-anarchy-smorgasbord-cn.netlify.app/)

> ✨ 无需安装，直接在浏览器中打开即可使用！

### 快速开始

#### 第一步：选择你感兴趣的类别

打开应用后，你会看到 28 个关系类别：

| 💕 浪漫 | 💭 情感亲密 | 🙏 宗教/灵性 | 💰 财务 |
| 💓 性 | 💬 沟通频率 | 🏥 照护 | 📊 权力/等级 |
| 🫂 身体亲密 | 📞 沟通方式 | ❤️ 情感支持 | 🏷️ 标签/称谓 |
| 🏠 生活日常 | 🎭 权力互动与性癖 | 🤝 伙伴关系 | ⚖️ 边缘化、特权、结盟与倡导 |
| ⏰ 共度时光 | 👥 陪伴 | ❓ 需要讨论的事项 | 👨‍👩‍👧 共同照护 |
| 🎯 未来规划 | 💻 科技 | 🔒 排他性 | 💼 职业/工作 |
| 🤝 协作 | 🎨 创作 | 👀 公开表达亲密 | ⚖️ 法律 |

**操作方式**：
- 点击类别卡片选中
- 可以多选
- 点击"随机抽取"快速选择 3 个类别
- 选好后点击"开始探索"

#### 第二步：创建关系档案

如果是第一次使用，需要创建一个关系档案：

1. 填写**你的名字**（例如：张三）
2. 填写**对方名字**（例如：李四）
3. 选择**关系标签**（例如：伴侣、朋友、QPR、选择性家人等，或自定义）

**提示**：
- 档案名称可以不填，系统会自动生成"张三 & 李四"
- 可以创建多个档案，对应不同的人

#### 第三步：填写卡牌

进入卡牌讨论页后，你会看到该类别中的每张卡牌：

**💡 重要提示**

这个工具高度依赖于你与伙伴之间的**互动和沟通**，它不会独立于关系而存在。

- ⚠️ 不同的人对同一个词汇或描述可能有不同的定义
- 💬 请勇敢地用你自己的理解与伙伴一起讨论
- 🤝 这个工具只是起点，真正的对话才是核心
- ✨ 没有标准答案，只有你们共同定义的答案

---

**卡牌内容**：
- 每张卡牌都有一个问题或描述
- 中英文对照显示
- 有些卡牌会有上下文问题（引导性问题）

**示例**：
- "我们多久见一次面？"
- "什么样的身体接触让我们感到舒服？"
- "我们如何看待金钱和财务责任？"

**选择你的状态**（可多选）：

| 状态 | 说明 |
|------|------|
| ✅ 同意 | 同意该描述或期待 |
| 🔥 必要 | 对你来说是必要的 |
| 💭 也许 | 可能有兴趣或不确定 |
| 🔮 未来可能 | 现在不行，但未来可能考虑 |
| ❓ 需要讨论 | 想要和伙伴进一步讨论 |
| ❌ 绝对不行 | 明确表示不同意 |
| 🚫 硬性界限 | 不可触碰的底线 |

**提示**：
- 这些状态可能不能直接回应关键词或问题，这很正常
- 鼓励你和伙伴发散讨论，探索更多可能性
- 重要的是对话本身，而非一定要达成明确的结论

---

**添加备注**（可选）：
- 记录你的想法、偏好或特殊要求
- 帮助你记住细节

**导航**：
- 点击"跳过"：跳过当前卡牌
- 点击"保存并继续"：保存并进入下一张
- 完成一个类别后，会自动跳到下一个类别

#### 第四步：查看档案详情

完成填写后，可以在档案详情页查看：

- 档案基本信息
- 探索进度
- 所有已填写的项目
- 导出档案（保存为 TXT 文件）
- 分享口令（让对方导入你的档案）

#### 第五步：与伙伴协作

这个工具的核心在于**关系中的互动和沟通**，它不会独立于关系而存在。无论你选择哪种协作方式，重要的是与你的伙伴一起讨论、理解和定义这些词汇。

**重要提醒** 💡
- 不同的人对同一个词汇可能有不同的定义
- 请勇敢地用你自己的定义与伙伴一起讨论
- 这个工具只是起点，真正的对话才是核心
- 没有标准答案，只有你们共同定义的答案

---

**协作方式对比**

| | 异步协作 | 同步协作 |
|---|---|---|
| **操作步骤** | 1. 完成你的档案填写<br>2. 复制分享口令给伙伴<br>3. 伙伴导入并填写<br>4. 对比档案查看差异 | 1. 双方一起打开应用<br>2. 选择类别讨论<br>3. 现场填写状态和备注<br>4. 对比档案查漏 |
| **优势** | • 先独立思考，不受即时影响<br>• 有充足时间认真考虑<br>• 避免面对面回答的压力<br>• 适合异地或时间不匹配 | • 可即时讨论和澄清<br>• 立即看到对方反应<br>• 容易达成共识<br>• 增强连接感 |
| **劣势** | • 缺少即时反馈<br>• 需要更多时间达成共识<br>• 无法看到对方反应 | • 需要时间地点同步<br>• 可能受到即时压力<br>• 敏感话题需要更多思考时间 |

**建议**：
- 两种方式可以结合使用
- 敏感话题可以先用异步方式思考，再同步讨论
- 重要的是找到适合你们的方式
- 无论哪种方式，保持开放和尊重的态度

#### 第六步：档案对比

选择两个或更多档案进行对比：

1. 在档案列表页勾选要对比的档案（2 个或更多）
2. 点击"对比"按钮
3. 查看你们的共识和分歧
4. 可以按类别筛选查看

**对比结果**：

| 结果类型 | 说明 |
|---------|------|
| ✅ 双方都同意 | 你们在这些方面达成共识 |
| 👤 只有你同意 | 你的期待与对方不同 |
| 🧑 只有对方同意 | 对方的期待与你不同 |
| ❌ 双方都不同意 | 双方都明确反对 |
